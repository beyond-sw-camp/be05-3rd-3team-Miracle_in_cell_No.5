<img width="536" alt="image" src="https://github.com/beyond-sw-camp/be05-3rd-3team-Miracle_in_cell_No.5/assets/134581020/b793a3b5-129c-431e-80f2-2ece190ecfc8">



## ⭐️ 프로젝트 소개

- SpringBoot와 Vue.js를 이용한 한화 시스템 Beyond SW Camp 5기 방명록 시스템<br>
- 사용자는 각자의 방명록을 가지고 서로의 방에 접속하여 방명록(칭찬,건의 등..)을 남길수 있는 미니 싸이월드<br>
- 솔방울이라는 토큰을 이용한 메인화면 확성기 사용
